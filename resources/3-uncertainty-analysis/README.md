# Direction 3: Uncertainty Analysis ⭐ Core

**[📁 Back to Main README](../../README.md)**

---

## 📖 Overview

Uncertainty analysis is the core research direction, focusing on quantifying and propagating various types of uncertainties to provide scientific basis for reliable decision-making.

---

## 📂 Sub-directions

### 3.1 Uncertainty Calibration
Model calibration, probability calibration, uncertainty calibration methods

### 3.2 Uncertainty Quantification in Deep Learning
Model uncertainty (epistemic), data uncertainty (aleatoric), Bayesian neural networks, MC Dropout

### 3.3 Uncertainty Fusion Methods
Multi-source uncertainty fusion, uncertainty propagation

### 3.4 Polynomial Chaos Expansion Methods
PCE, sparse PCE, adaptive PCE
